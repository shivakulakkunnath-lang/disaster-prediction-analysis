# disaster-prediction-analysis
Machine learning project for disaster prediction using python and data analysis techniques

## Project Overview
This project uses machine learning and data analysis techniques to predict disaster-related outcomes from the dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Prediction and Evaluation

## Files
- `disaster_prediction_ml.ipynb` → Main notebook containing complete analysis and model building
